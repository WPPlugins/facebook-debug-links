Provides links in your wordpress backend to easily open your page/post in the Facebook Debugger

== Description ==

This plugin provides links in your wordpress admin so you can easily check your page/post in the Facebook Debugger.

It adds a metabox on the post/page edit screen and extra links in the row action part of the post/page manager.

The Facebook Debug Links are only displayed for published posts/pages.

You can also use it to clear your Facebook cache for posts/pages.

[Plugin homepage](http://xlino.com/projects/facebook-debug-link-wordpress-plugin/)

== Installation ==

= Installation =

 1. Upload the plugin directory to the `/wp-content/plugins/` directory or use the installer via backend of WordPress
 1. Activate the plugin through the 'Plugins' menu in WordPress
